---
description: How to run the Student Data Management System
---

1. Ensure XAMPP (or similar) is installed.
2. Start Apache and MySQL from the XAMPP Control Panel.
3. Navigate to `http://localhost/phpmyadmin`.
4. Run the SQL commands found in `database.sql` to setup the database.
5. Verify that `api/db_connect.php` has the correct MySQL credentials.
6. Place the project folder in `C:\xampp\htdocs\`.
7. Open `http://localhost/[folder_name]/index.html` in your browser.
